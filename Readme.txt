数据结构：使用了结构体 Number 来表示带有颜色标记的元素，以及结构体 Config 来存储用户设置。

动态内存分配：使用 malloc 和 realloc 来动态分配和扩展结构体数组的内存空间。

随机数生成：利用 rand 和 srand 生成随机数，用于创建随机数据和随机选择启动画面。

时间测量：使用 clock 函数来测量排序算法的执行时间。

文件操作：实现了将结构体数组保存到文件和从文件中读取结构体数组的功能，以及保存用户设置到文件的功能。

排序算法：实现了冒泡排序、选择排序、插入排序、希尔排序、快速排序、堆排序等排序算法，并支持单步显示排序过程、比较次数和交换次数显示、运行时间显示、排序结果显示等功能。

交互界面：使用 ANSI 转义码实现了彩色文本输出，提升了用户交互体验。

彩蛋功能：添加了猴子排序作为彩蛋功能，当数据未排序时，程序会自动进行猴子排序直到数据有序。

设置功能：用户可以根据需要自定义是否显示排序过程、比较次数和交换次数、运行时间、排序结果，以及是否保存数据。
